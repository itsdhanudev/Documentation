# Vector3.ONE

Represents `(1, 1, 1)`.

## Method Signature

```java
Vector3.one()
```

## Description

Common default for unit scale in 3D transforms.

## Mathematical Definition

`ONE = (1,1,1)`

## Parameters

- None.

## Returns

- `Vector3`: vector of ones.

## Mutability

- Treat as immutable constant pattern.

## Edge Cases

- None beyond floating-point representation.

## Usage Example

```java
myTransform.setScale(Vector3.one());
```

## Mathematical Example

`v.multiply(ONE) = v`
