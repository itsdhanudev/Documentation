# Vector3.Vector3()

Creates a default `Vector3`.

## Method Signature

```java
Vector3()
```

## Description

Builds a new vector with engine default component values (typically `0,0,0`).

## Parameters

- None.

## Returns

- `Vector3`: new default instance.

## Mutability

- Constructor initializes a new object only.

## Edge Cases

- No edge-case parameters.

## Usage Example

```java
Vector3 velocity = new Vector3();
velocity.lerp(new Vector3(0f, 0f, 12f), 0.2f);
```

## Mathematical Example

Default value is typically `(0,0,0)`.
