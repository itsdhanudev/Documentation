# UI3DInputSystem

Habilita interacao de UI em superfices 3D usando toque na tela.

## Criar no Editor

1. Garanta que existe um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `
UI3DInputSystem
`.

## Como Usar

1. Pegue o componente com `myObject.findComponent(UI3DInputSystem.class)`.
2. Leia e altere as propriedades do componente para controlar o comportamento.
3. Teste o input ou layout no modo Play.

## Exemplo em Java

```java
SpatialObject myObject = /* your object */;
UI3DInputSystem comp = myObject.findComponent(UI3DInputSystem.class);
if (comp != null) {
    // set the maxHitDistance value:
    comp.maxHitDistance = 1f;
    // read the maxHitDistance value:
    float maxHitDistanceValue = comp.maxHitDistance;

}
```
