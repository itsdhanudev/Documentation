# UIAspectRatio

Mantem a proporcao largura/altura para evitar distorcao.

## Criar no Editor

1. Garanta que existe um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `
UIAspectRatio
`.

## Como Usar

1. Pegue o componente com `myObject.findComponent(UIAspectRatio.class)`.
2. Leia e altere as propriedades do componente para controlar o comportamento.
3. Teste o input ou layout no modo Play.

## Exemplo em Java

```java
SpatialObject myObject = /* your object */;
UIAspectRatio comp = myObject.findComponent(UIAspectRatio.class);
if (comp != null) {
    // set the mode value:
    comp.mode = new Mode();
    // read the mode value:
    Mode modeValue = comp.mode;

    // set the proportion value:
    comp.proportion = 1f;
    // read the proportion value:
    float proportionValue = comp.proportion;

}
```
