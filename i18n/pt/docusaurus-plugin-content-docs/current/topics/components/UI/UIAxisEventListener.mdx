# UIAxisEventListener

Recebe valores de eixo de controles como joystick e volante.

## Criar no Editor

1. Garanta que existe um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `
UIAxisEventListener
`.

## Como Usar

1. Pegue o componente com `myObject.findComponent(UIAxisEventListener.class)`.
2. Leia e altere as propriedades do componente para controlar o comportamento.
3. Teste o input ou layout no modo Play.

## Exemplo em Java

```java
SpatialObject myObject = /* your object */;
UIAxisEventListener comp = myObject.findComponent(UIAxisEventListener.class);
if (comp != null) {
    // set the axisName value:
    comp.axisName = "value";
    // read the axisName value:
    String axisNameValue = comp.axisName;

    // set the value value:
    comp.value = new Vector2(1, 1);
    // read the value value:
    Vector2 valueValue = comp.value;

}
```
