# UIHorizontalScrollView

Container com scroll horizontal que recorta e move os filhos.

## Criar no Editor

1. Garanta que existe um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `
UIHorizontalScrollView
`.

## Como Usar

1. Pegue o componente com `myObject.findComponent(UIHorizontalScrollView.class)`.
2. Leia e altere as propriedades do componente para controlar o comportamento.
3. Teste o input ou layout no modo Play.

## Exemplo em Java

```java
SpatialObject myObject = /* your object */;
UIHorizontalScrollView comp = myObject.findComponent(UIHorizontalScrollView.class);
if (comp != null) {
    // set the scroll value:
    comp.scroll = 1;
    // read the scroll value:
    int scrollValue = comp.scroll;

}
```
