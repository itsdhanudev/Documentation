# UIRadioGroup

Define um grupo para que apenas um UIRadioButton fique marcado.

## Criar no Editor

1. Garanta que existe um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `
UIRadioGroup
`.

## Como Usar

1. Pegue o componente com `myObject.findComponent(UIRadioGroup.class)`.
2. Leia e altere as propriedades do componente para controlar o comportamento.
3. Teste o input ou layout no modo Play.

## Exemplo em Java

```java
SpatialObject myObject = /* your object */;
UIRadioGroup comp = myObject.findComponent(UIRadioGroup.class);
if (comp != null) {
}
```
