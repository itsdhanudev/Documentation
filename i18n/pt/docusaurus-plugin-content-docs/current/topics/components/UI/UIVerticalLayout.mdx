# UIVerticalLayout

Organiza filhos em coluna vertical com espacamento e alinhamento.

## Criar no Editor

1. Garanta que existe um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `
UIVerticalLayout
`.

## Como Usar

1. Pegue o componente com `myObject.findComponent(UIVerticalLayout.class)`.
2. Leia e altere as propriedades do componente para controlar o comportamento.
3. Teste o input ou layout no modo Play.

## Exemplo em Java

```java
SpatialObject myObject = /* your object */;
UIVerticalLayout comp = myObject.findComponent(UIVerticalLayout.class);
if (comp != null) {
    // set the spacing value:
    comp.spacing = 1;
    // read the spacing value:
    int spacingValue = comp.spacing;

}
```
