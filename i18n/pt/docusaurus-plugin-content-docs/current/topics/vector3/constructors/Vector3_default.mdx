# Vetor3.Vector3()

Creates a default `Vector3`.

## Method Signature

```java
Vector3()
```

## Descrição

Builds Um novo vector with engine default component values (typically `0,0,0`).

## Parameters

- None.

## Retorna

- `Vector3`: new default instance.

## Mutability

- Constructor initializes Um novo object only.

## Edge Cases

- No edge-case parameters.

## Usage Example

```java
Vector3 velocity = new Vector3();
velocity.lerp(new Vector3(0f, 0f, 12f), 0.2f);
```

## Mathematical Example

Default value is typically `(0,0,0)`.
